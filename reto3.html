Kata time!

Crear una clase en Javascript llamada "Animal" que toma los argumentos "name" y "type". Debería tener un método "toString" que retorne una cadena de texto indicando la información de los argumentos suministrados en una forma legible. Debería permitir que la propiedad "name" sea seteable, lo siguiente es un ejemplo de cómo se usaría el código final: 

var dog = new Animal('Max', 'dog');

dog.toString(); // should return 'Max is a dog'
dog.type; // should == 'dog'
dog.name; // should == 'Max'
dog.name = 'Lassie'; // should set name to 'Lassie'
